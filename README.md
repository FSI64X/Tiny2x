Tiny2x | Image Upscaling for Everyone

Not everyone has a powerful PC, and that shouldn't stop you from upscaling your images.
Tiny2x is a lightweight, free image upscaler built specifically for low-end computers. No GPU required, no sign-up, no limitations. Just download and start upscaling.
Minimum specs: 4GB RAM · 4-core CPU
Recommended specs: 8GB RAM · 4–6 core CPU
Price: Free. Forever.

MANUAL:

• Download the .ZIP file of the Tiny2x software from the "releases" tab.
• Extract the Tiny2x.zip file.
• Run the "Tiny2x.exe" file.

-----------------------------------

In the software, select the image you want to upscale or drag-and-drop it into the area then hit "Upscale".
Depending on the image and your CPU, after a moment your image will be upscaled and the upscaled image should be where your original image is. You can set a custom folder in the "Settings" tab (click the gear icon to open the settings tab) where you want your upscaled image to be.
And that's how you can upscale your image with Tiny2x! Fast & Easy!

Tile Size and Tile Overlap/Padding:

• Tile Size: Tiling splits your image into smaller chunks so your CPU can process each tile one by one without straining itself. Tile Size is the pixel dimension of that tile meaning that smaller tile size (e.g. 64px) will use less RAM whereas bigger tile size (e.g. 256, 512px) will use more RAM. If you device is not that capable then use smaller tile size.

• Tile Overlap: Tile Overlap adds an overlap border on each tile before processing it, without it, you'd see visible seams or grid lines where the tiles meet. Higher tile overlap value = smoother seams, less overlap value = some visible seams. Keep in mind, Higher Tile Overlap uses more computation power. Generally,   32px-64px is safe default. You can tweak it according to your device.
